Welcome to the Islamic Prayer Times App

Get accurate prayer times based on your location!

About the Project

This project provides Islamic prayer times based on your city and country. You can get prayer times for today or the whole month. It also allows you to save the prayer times to a CSV file for later reference.

How to Use

  • Enter the name of your city and country.
  • Choose whether you want prayer times for today or the whole month.
  • The app will display the prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha).
  • You can save the times to a CSV file if needed.

Installation & Usage

To run the app locally, follow these steps:

  1. Clone the repository: git clone https://github.com/mostafa19ashry/islamic_calendar_scraper.git
  2. Navigate to the project folder: cd islamic_calendar_scraper
  3. Install the required dependencies: pip install -r requirements.txt
  4. Run the project: python main.py
